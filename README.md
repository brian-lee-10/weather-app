# Weather App

A simple Python application that allows users to fetch current weather and a 3-hour interval forecast for any city using the OpenWeatherMap API.

## Features

- Fetches current weather data, including temperature and weather description, for a specified city.
- Displays a 3-hour interval forecast for the next 24 hours.
- Easy-to-use graphical user interface (GUI) built with Tkinter.

## Requirements

- Python 3.x
- `requests` library for making HTTP requests
- OpenWeatherMap API key
